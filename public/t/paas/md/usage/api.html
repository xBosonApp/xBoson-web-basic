数据元，数据集查询/导出API

API：

          app：c879dcc94d204d96a98a34e0b7d75676
          mod：mm
          api：getdatatable

参数：

          typecd：类别编码
          
            数据元查询参数：
          decd：数据元编码
          en：英文名称
          cn：中文名称
          datatype：数据类型
          status：状态
          
            数据集查询参数
          decd：数据元编码
          en：英文名称
          cn：中文名称
          must：是否必填{1,0}
          mk：是否主键{1,0}
          
          pagenum：分页查询（非必需）
          pagesize：分页查询（非必需）
          
          _is_download：是否下载（非必需）{1,0}
          type：下载类型（非必需）{1:CSV,2:JSON,3:Excel}
          charset：字符集（非必需）
