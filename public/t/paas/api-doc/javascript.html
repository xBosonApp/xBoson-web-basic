JavaScript开发规范：
代码中采用'// todo:说明'做临时标记，JavaScript代码内字符串全部统一采用单引号; 
<div class="picture"><img src="img.jpg" /></div>
var $img = $('.picture').find('img'); 或 var $img = $('img', '.picture');
经过测试发现前者在性能上比后者稍微快些，查看 jQuery 源码可发现后者会转成find方法，在转换时要花点时间。但当我们封装一个常用的方法时，使用 context 会比较合适（会提高我们的代码质量），如下： 
reSize = function (div) {
  $('img', div).width(100`);
  $(div).find('img').width(100);
}
HTML开发规范：
1、标签元素属性[id]值具有全局唯一特性，需要考虑页面loadHTML模式加载后的[id]值不能重复，建议附加页面名称做前缀；
2、一个页面内可以有多个[form]表单，但不能互相嵌套[form]表单，注意：[form]表单内的标签元素属性[name]值不能重复；
3、标签元素属性的[id]、[class]值对大小写敏感，多个单词组合时统一采用'-'连接符分割；
