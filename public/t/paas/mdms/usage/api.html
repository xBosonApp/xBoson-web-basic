数据字典查询/导出API

API：

          app：d2c8511b47714faba5c71506a5029d94
          mod：DataDict
          api：getdatatable

参数：

          typecd：类别编码
          
            数据字典查询参数：
          version：版本
          dictcd：字典码
          dictnm：字典名称
          shortkey：快捷码
          status：状态
          
          pagenum：分页查询（非必需）
          pagesize：分页查询（非必需）
          
          _is_download：是否下载（非必需）{1,0}
          type：下载类型（非必需）{1:CSV,2:JSON,3:Excel}
          charset：字符集（非必需）
